Autonomous Drone Framework 🚁
<img src="docs/assets/drone-main-margin.png" align="right" width="400px" alt="Autonomous Drone">
This project is an open-source framework for creating autonomous drones, built on top of ROS (Robot Operating System) and compatible with PX4 autopilot firmware. It provides user-friendly tools for controlling drones and is designed to work seamlessly with Raspberry Pi as a companion computer. Once you've installed Raspberry Pi on your drone and flashed the preconfigured image to its microSD card, getting your drone airborne is quick and straightforward.

The framework is ideal for educational purposes, research, and hobbyist projects. It includes support for computer vision, sensor integration, and autonomous navigation, making it a powerful tool for developing advanced drone applications.

The main documentation is available at https://docs.drone-framework.com.

Raspberry Pi Image
A preconfigured Raspberry Pi image is available for easy setup. This image includes all the necessary software and configurations to get your drone up and running quickly.
